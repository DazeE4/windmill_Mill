Privacy Policy

Windmill Automation  ·  Last updated: March 2026

This application is a personal automation tool operated by a single individual for personal passive income purposes. It does not collect, sell, or share personal data of any third party.

1. Who We Are

This privacy policy applies to the Windmill Automation application ("the App"), a personal automation system built on Windmill Cloud (windmill.dev). The App is operated by a single individual for personal use to automate content publishing and digital product delivery. It is not a commercial service offered to others.

2. What the App Does

The App performs the following automated tasks using third-party platform APIs:

  • Reads publicly available trending topics from Reddit's public API (read-only, no account access)
  • Reads publicly available news headlines from NewsAPI
  • Publishes content to dev.to using the operator's own account credentials
  • Posts pins to Pinterest using the operator's own business account
  • Uploads product images to Printify using the operator's own account
  • Posts short videos to YouTube and Instagram using the operator's own accounts
  • Delivers digital products to buyers who purchase through Gumroad
  • Sends scheduled email newsletters via SendGrid to opted-in subscribers
  • Generates content using the Groq AI API (no personal data is sent to Groq)

3. Data We Collect

The App collects the minimum data necessary to operate each automated function. No data is collected from visitors to this privacy policy page.

Buyer data (Gumroad purchases only):

  • Email address — used solely to deliver the purchased digital product
  • Full name — used solely to personalise the delivery email
  • Product ID and sale ID — used for delivery confirmation and record-keeping
  • Purchase price — used for income tracking in a private Google Sheet

Newsletter subscriber data:

  • Email address — collected only from buyers who have already purchased a product via Gumroad. No separate opt-in list is maintained. Subscribers may unsubscribe at any time by replying to any newsletter with "unsubscribe".

Data we do NOT collect:

  • No browsing data or cookies from website visitors
  • No Reddit user data — the App only reads public subreddit post titles (public data)
  • No Pinterest user data — the App only writes to the operator's own boards
  • No Instagram user data — the App only posts to the operator's own account
  • No payment card data — all payment processing is handled entirely by Gumroad
  • No IP addresses of any visitors or customers

4. How We Use Data

Buyer email addresses and names are used exclusively for:

  • Delivering the purchased digital product (one-time email with download link)
  • Sending weekly content newsletters (opted-in buyers only, unsubscribe available)
  • Internal income tracking in a private Google Sheet accessible only to the operator

We do not use personal data for advertising, profiling, or any purpose other than fulfilment of the purchased product and the newsletter service.

5. Data Sharing

We do not sell, rent, or trade personal data to any third party under any circumstances.

Personal data passes through the following processors solely to fulfil delivery:

  • Gumroad — payment processing and purchase record (gumroad.com/privacy)
  • SendGrid (Twilio) — email delivery of products and newsletters (sendgrid.com/privacy)
  • Cloudflare R2 — secure storage of digital product files (cloudflare.com/privacypolicy)
  • Google Sheets — private income tracking spreadsheet, operator access only (policies.google.com/privacy)

No personal data is shared with Reddit, Pinterest, Instagram, Printify, dev.to, YouTube, Groq, or any other platform used by the App.

6. Reddit API Compliance

The App uses the Reddit API in strict compliance with Reddit's Responsible Builder Policy.

  • Only public subreddit data is accessed (post titles and scores — no private messages, no user data)
  • API requests are made at a rate well below Reddit's published limits (~30 requests/day vs 100/minute allowed)
  • No Reddit account data, credentials, or personal information is stored
  • The App does not scrape Reddit outside of official API endpoints
  • The App does not manipulate votes, comments, or any Reddit content
  • The App does not impersonate any Reddit user or subreddit
  • Data retrieved from Reddit is used only to identify trending topics — no Reddit content is republished verbatim

7. Pinterest API Compliance

The App uses the Pinterest API v5 to post content to the operator's own Pinterest boards. It complies with Pinterest's Developer Guidelines.

  • Only the operator's own account is accessed — no Pinterest user data is read or stored
  • All pins posted are original content created by the automation system
  • No Pinterest user data, follower data, or engagement data is collected
  • Affiliate links in pin descriptions are disclosed as required by applicable regulations

8. Data Retention

  • Buyer email addresses are retained in the Windmill KV store for up to 12 months for newsletter purposes
  • Buyers may request deletion of their email at any time by contacting the operator (see Section 11)
  • Income records in Google Sheets are retained for personal tax and accounting purposes
  • Digital product files stored in Cloudflare R2 are retained indefinitely unless the product is discontinued
  • No purchase history or personal data is retained beyond what is described above

9. Data Security

All API credentials are stored as encrypted secret variables within Windmill Cloud. Digital product files are stored in Cloudflare R2 with access controlled by short-lived pre-signed URLs (expire after 1 hour). No personal data is stored in plain text. The operator's Google Sheet is accessible only via service account credentials, not publicly accessible.

10. Your Rights

If you have purchased a product and would like to exercise any of the following rights, contact the operator (Section 11):

  • Access: Request a copy of personal data held about you
  • Deletion: Request removal of your email from the newsletter list
  • Correction: Request correction of inaccurate data
  • Portability: Request your data in machine-readable format
  • Unsubscribe: Reply "unsubscribe" to any newsletter email — immediate removal

Requests will be responded to within 30 days.
